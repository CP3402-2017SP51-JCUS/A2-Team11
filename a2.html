Goal/s

The goal of the website is to sell ice-cream. Our main product is the DIY Cream Baby, which is a product that can be personalized by the client to fit his own tastes.

We created a simple design so that the user would find the navigation easy. The goal is to keep the client on the website until he buys a product.

Comments

The staging state is not similar to the production site but it helped us to choose which pluggins we wanted to use (in this case, woocommerce).

We used the plugin “form maker” in order to create the DIY ice-cream. Our theme didn’t support completely woo-commerce, so the options to add attributes to the product were not available. Hence, we used a form to create a new product that the client can create.

We added another plugin, Meta Slider, for the homepage to add an image gallery that would show the images with a sliding motion.
